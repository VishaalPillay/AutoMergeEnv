# AutoMergeEnv
A real-world OpenEnv benchmark challenging AI agents to resolve complex, semantic Git merge conflicts. Featuring deterministic Pytest graders and lightweight state management, it evaluates an agent's ability to safely integrate multi-branch code within strict Hugging Face Space constraints
